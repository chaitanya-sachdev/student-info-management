# Student Information Management App

A Flutter mobile application integrated with MySQL backend for managing student information.

## Features
- Student Registration Form
- Store student data in MySQL database via PHP APIs
- Retrieve and display all student records

## Tech Stack
- **Frontend**: Flutter, Dart
- **Backend**: PHP, MySQL
- **Server**: XAMPP

## Setup Instructions

### Backend Setup
1. Start XAMPP Server
2. Place PHP files in htdocs folder
3. Import SQL script to create database

### Frontend Setup
1. Install Flutter SDK
2. Run `flutter pub get`
3. Update API endpoint in code
4. Run `flutter run`
